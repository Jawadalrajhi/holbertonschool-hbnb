# HBnB Evolution

## Overview
HBnB Evolution is a simplified AirBnB-like application developed as part of the Holberton School curriculum.  
The project focuses on designing and documenting a scalable software architecture before moving into the implementation phases.

This repository contains **Part 1: Technical Documentation**, which defines the system architecture, business logic design, and interaction flows using UML diagrams.

---

## Project Scope
The main goal of this project is to:
- Design a clear layered architecture
- Define core business entities and their relationships
- Illustrate system interactions through sequence diagrams
- Provide technical documentation to guide future implementation

No application code is written in this phase. The focus is strictly on **design and documentation**.
